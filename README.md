## Installation
On Raspberry Pi, there'll need an SD with Raspbian. 
It's best to update it.
```
sudo apt-get update
```
An FTP server must be installed.
```
sudo apt-get install proftpd
```
There'll need the pygame library to run the script.
```
pip3 install pygame
```
